Das [jüngste Release](https://github.com/Schachigel/DKV2/releases/tag/0.20.0.11) ist 0.20.0.11. Es ist das letzte Release, das mit Qt 5 und qmake gebaut wird. In kürze erscheint ein Update, das auf Qt 6 aufsetzt und cmake für die Erstellung der Software verwendet.

Der aktuelle Linux Build wird per Github Action erstellt. 
Der Windows Build und die Erstellung der Installation wurde vmo Autor der Software erstellt. 

Mit dem nächsten Release werden Linux, Mac und Windows Versionen zur Verfügung stehen, die alle automatisiert mit Github Actions gebaut wurden.
